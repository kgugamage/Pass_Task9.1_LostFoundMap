# Pass Task 9.1 - Lost and Found Map App

This is an Android Lost and Found app with map features.

## Features

- Create lost or found advert
- Add item name, category, description and image
- Get Current Location button
- Save item details using SQLite
- View all saved items
- Show saved items on a map
- Radius search on the map screen

## How it works

The user creates an advert and enters the item details. The location is added using the Get Current Location button. The item is saved into the SQLite database.

The Show On Map button opens the map screen. Saved items are shown as markers on the map. The user can also enter a radius in kilometres and search nearby items.

## Technologies used

- Android Studio
- Java
- SQLite
- WebView map
